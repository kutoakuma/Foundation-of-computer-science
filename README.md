# Foundation-of-computer-science
## Effective data management and security are major issues in contemporary computer systems. Three related tasks that examine encoding formats, computational complexity, and database normalization are used in this coursework to address these issues. When taken as a whole, these exercises show how fundamental computer science ideas can be applied to real-world data management and security problems.
## Task1: Enhancing Secure Data Exchange with Encoding Formats and Protocol Integration. It involves exploring encoding formats including Base64, ASCII, Hex and URL encoding, examining their role in securing data exchange throughout web protocol.
## Task2: Classroom Seating Arrangement Problem. This task explores the seating arrangement problem to demonstrate fundamental concept in computational complexity theory. 
## Task3: College Club Membership Management. In this task the normalization o f a college club membership management scenario.
